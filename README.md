## 📖 简介 (Introduction)

这是专业的加密货币配资交易系统。它旨在帮助平台方快速搭建自己的数字资产配资平台，支持用户使用小额本金例如1:10配资，撬动大额资金进行交易（杠杆交易），提供完整的风控体系、自动强平机制以及多级代理返佣系统。

## ✨ 核心特性 (Features)

- � **杠杆配资**：支持 1-100 倍杠杆自由配置，用户存入保证金即可获得配资额度进行交易。
- 💰 **多币种支持**：原生支持 USDT、BTC、ETH 等主流加密货币的充值与提现，自动对接交易所行情。
- ⚡ **极速交易**：对接主流交易所（Binance/OKX）深度，实现毫秒级下单，不卡顿、不滑点。
- 🛡️ **智能风控**：
  - **自动强平**：当亏损达到预警线或平仓线时，系统自动触发强平，保障平台资金安全。
  - **穿仓保护**：多重风控机制，防止极端行情下用户穿仓。
- 📊 **代理商系统**：支持无限级代理模式，代理商可获得用户交易手续费、利息分红等多重返佣。
- � **利息管理**：支持按天、按周、按月收取配资利息，费率可自定义配置。
- � **多端支持**：提供 iOS、Android APP 以及 H5 网页端，数据实时同步。

## 🛠️ 技术栈 (Tech Stack)

- **后端框架**: ThinkPHP 6.0
- **数据库**: MySQL 5.7 / 8.0
- **缓存**: Redis (用于行情推送与高频交易)
- **运行环境**: PHP 8.0+ (推荐 8.1), Nginx
- **前端框架**: UniApp (Vue.js)
